# Web Applications Repository

A curated collection of production-style web applications built to demonstrate practical skills in frontend, backend, databases, authentication, and deployment using the MERN/PERN stack. Each project focuses on real-world architecture, clean code practices, and complete feature implementation suitable for professional web development.

## 📁 Projects

```
web-applications/
├── portfolio-website-template/     # Next.js portfolio template
├── fiverr-web-application/         # Fiverr clone project
├── finance-dashboard/              # React finance dashboard
└── README.md                      # This file
```

## 🚀 Projects

### 1. Portfolio Website Template
Modern, responsive portfolio template built with Next.js 14 and React 18.

**Tech Stack:**
- Next.js 14 (App Router)
- React 18
- Pure CSS (no UI libraries)
- JavaScript

**Features:**
- Responsive design
- Smooth scrolling navigation
- Project showcase
- Skills section
- Contact form

### 2. Fiverr Web Application
Full-featured freelance platform clone with React frontend and Node.js backend.

**Tech Stack:**
- React (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)
- Authentication & Authorization

**Features:**
- User authentication
- Service listings
- Order management
- Reviews and ratings

### 3. Finance Dashboard
A modern financial dashboard built with React and Tailwind CSS for tracking income, expenses, and financial insights.

**Tech Stack:**
- React 19
- Tailwind CSS v4
- Recharts (for charts)
- Zustand (state management)
- Vite (build tool)

**Features:**
- Real-time financial summaries
- Interactive charts and graphs
- Dark mode support
- Responsive design
- Transaction tracking
- Financial insights

## 📱 Usage

Each project is contained in its own folder with complete source code and documentation.

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sahil-kate/web-applications.git
   cd web-applications
   ```

2. **Navigate to a project**
   ```bash
   cd portfolio-website-template
   # or
   cd fiverr-web-application
   # or
   cd finance-dashboard
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

## 🛠️ Technologies Used

- **Frontend**: React, Next.js, HTML5, CSS3, JavaScript, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose
